# Software Development Processes Powered by AI Agents

A hands-on course exploring how AI agents can automate and enhance every phase of the software development lifecycle — from Git workflows and architecture design to CI/CD pipelines and test-driven development.

## Chosen Kata

For the Software Architecture module, the selected system is based on the Minesweeper kata.

Minesweeper is a classic grid-based game where the player uncovers cells while trying to avoid hidden mines. Each cell provides information about nearby mines, and empty cells automatically reveal surrounding areas.

This problem is a good candidate for architectural design because it includes multiple responsibilities such as generating the game board, handling user actions, maintaining game state, and evaluating win or loss conditions.
These responsibilities can be separated into distinct components, which makes the system suitable for applying the arc42 template and visualizing the structure using the C4 model.
